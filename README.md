# Fraud Detection App

A **Streamlit web app** for detecting fraudulent transactions using a pre-trained machine learning model. Users can input transaction details, and the app predicts whether the transaction is **legitimate** or **fraudulent**, along with the probability risk.

---

## Author

Developed by **Adebimpe0**

---

## Features

- User-friendly **Streamlit interface**  
- Input transaction details: value, account info, product, channel, etc.  
- Predicts fraud using a **pre-trained ML model**  
- Handles **categorical and numeric features** with encoding and scaling  
- Displays **prediction result and fraud probability**
